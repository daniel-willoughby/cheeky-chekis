Cheki placeholder photos
========================

Save your decorated cheki photos in THIS folder with these exact names:

  cheki-01.jpg   (the red/black qipao one)
  cheki-02.jpg   (the strawberry lolita one)
  cheki-03.jpg   (the KITTY BEL / Hello Kitty one)
  cheki-04.jpg   (the pink-bow / nai-te-chan one)

The app cycles these three across the seeded chekis.
.jpg, .jpeg, .png all work as long as the name matches (see REAL_CHEKIS in
src/data/chekiArt.ts if you want to change names or add more).

If a file is missing, the app falls back to placeholder.svg so nothing breaks.
